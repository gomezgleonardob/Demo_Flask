\section{Número par}

Escriba la función \lstinline!par(x)! que retorne \lstinline!True! si
\lstinline!x! es par, y \lstinline!False! si es impar:

\begin{lstlisting}
>>> par(16)
True
>>> par(29)
False
>>> par('hola')
Traceback (most recent call last):

  File "<console>", line 2, in par

\end{lstlisting}
